万物的形状
都在重复

Structural 是一个跨领域思维引擎。它基于一个反直觉的观察：看似完全无关的现象，在数学结构层面往往是同一件事。放射性衰变和药物代谢、股价波动和热力学涨落、传染病传播和谣言扩散——它们在各自领域的语言下毫无关系，但背后是同一个方程。

Goal

整体目标 · 把"发现跨域同构"系统化

科学史上的一些关键突破——Shannon 用热力学的熵建立信息论、Black-Scholes 把热传导方程搬到期权定价、模拟退火把冶金过程变成组合优化——都是"同一个数学骨架第一次被搬到另一个领域"的瞬间。传统上这类跨域借用依赖极少数通才的偶然洞察。

Structural 想做的是：把这种发现过程从偶然变成可搜索、可复现、可验证的流程。四代管道（V1 / V2 / V3 / V4）是同一个目标的不同切面——从 embedding 召回，到结构代数匹配，到普适类组织和实证验证。产出的每一个候选都带数学方程、变量映射、目标期刊、执行计划，是领域专家可以直接拿去写论文的"种子"。

Why

为什么这有用

当你在自己的领域里遇到一个难题，答案也许根本不在你的领域里。在另一个和你完全无关的学科，可能有人已经解决了同一个数学问题——他们的术语不同、场景不同，但数学骨架完全一致。Structural 让你跨越学科的围墙，直接借用别人的答案。

Claude Shannon 借用热力学的"熵"奠定了信息论。Black 和 Scholes 把热传导方程搬到金融市场上得到了期权定价公式。模拟退火算法把冶金学的物理过程变成了组合优化的利器。这些跨领域的突破不是偶然——它们都是结构同构被发现的瞬间。Structural 想做的是把这种"发现"系统化。

How

它是怎么工作的

传统的搜索工具基于语义或关键词匹配——它找的是"看起来像"的东西。Structural 不一样。它的核心是让机器识别"数学结构相似"，而不是"文字表面相似"。为此我们建了三条并行的检索管道——它们从不同角度寻找同一件事，最后合并去重。

V1 · 广义 embedding（covering recall）

用 text2vec-base-chinese 作为 backbone，在 84 种结构类型 × 1,214 条人工标注样本上做对比学习微调（MultipleNegativesRankingLoss）。目标是让"同一结构的不同领域描述"在 768 维空间中互相靠近，"不同结构"互相推开。训练完成后 Silhouette 0.85、Retrieval@5 100%。

路径：自然语言描述 → V1 embedding → cosine 相似度 → top-k。
优势：召回广（在 4,443 KB 上找到 339,913 对高相似跨域配对）。
缺点：精度一般，"共享关键词"和"共享数学"区分不够。
角色：宽撒网——找潜在候选，不保证质量。

V2 · 严格 embedding（precision retrieval）

相同的 backbone，但训练集扩到 5,689 条（V1 原始 + 4,475 条新扩展），多了一倍的负样本和跨领域对。这让模型变得更"挑剔"——在同一个 4,443 KB 上只产出 4,533 对高相似配对（严格度是 V1 的 75×）。

路径：自然语言 → V2 embedding → cosine → top-k。
评估：Silhouette 0.55（比 V1 在扩展集上更好）、Retrieval@5 96%。
角色：精度检索——同样是 embedding，但更能分辨浅层关键词相似。

V3 · StructTuple + LLM rerank（structural algebra）

前两代的 embedding 都是"统计相关"——把文本映射到向量空间看距离。V3 换了一个哲学：如果结构同构的本质是共享数学方程，就必须显式地表达这些方程，不是用相关性近似它。

我们用 Kimi/Opus 把 4,443 现象每一条都抽取成一个结构化元组（StructTuple），字段包括：

  • state_vars — 随时间变化的状态变量
  • dynamics_family — 动力学族：ODE1_exponential_decay / DDE_delayed_feedback / PDE_reaction_diffusion / Bistable_switch / Network_cascade / Phase_transition_1st 等 27 个枚举值
  • feedback_topology — positive_loop / negative_loop / delayed / bistable / none
  • boundary_behavior — runaway / saturation / limit_cycle / fold_bifurcation / phase_transition
  • timescale_log10_s — 特征时间的数量级
  • canonical_equation — 可推断的方程骨架（如 dx/dt = f(x(t-τ)) - g(x)）

检索时用字段级硬匹配：dynamics_family 不同直接拒绝；同族才进入细粒度评分。这过滤掉了"两个现象都含'反馈'、'阈值'但机制完全不同"的关键词陷阱。通过后再用 LLM pairwise rerank 做第二次过滤，每对输出 shared_equation 和 variable_mapping。

路径：自然语言 → LLM 推断 dynamics_family → StructTuple 字段匹配 → LLM rerank → top 候选 + 共享方程。
评估：在 1,000 候选上 LLM 5-score 密度 5.5%（V2 是 1.9%），深度分析 A 级命中率从 V2 的 ~20% 提升到 ~27%。
角色：结构代数——每个发现都带方程，直接可以写进论文的方法论段落。

三代并行，零重叠

最关键的发现：V1 的 24 个 tier-1、V2 的 19 个 A 级、V3 的 20 个 A 级发现，两两之间完全零重叠。这说明三条管道看到的是结构的不同切面——V1 看到"语义近邻"、V2 看到"精细语义"、V3 看到"数学同族"。三个并集 = 63 个独立的顶级跨域候选（数据集 SIBD-63）。

V4 · 普适类发现引擎（universality class engine）

V1–V3 产出的是离散的"A≅B"配对。但只要你仔细看 63 个 seed，会发现大量"A≅B 同时 A≅C"的结构——地震、S&P 股价、DeFi 清算、神经雪崩，它们两两都同构，但其实是在共享同一个更底层的数学物理规律。V4 的任务是把这些离散对子聚合成等价类（universality class）。

实现分五层：

  • Layer 1 · 图建模——把 SIBD-63 建成图，节点 = 领域现象，边 = 同构对；对语义相近的节点做别名归并（node_aliases.json，人工白名单 + LLM 建议）。
  • Layer 2 · 连通分量 + Louvain 社区——用 NetworkX 算 connected components 和 Louvain community，得到 23 个 candidate universality classes。
  • Layer 3 · LLM 精编——每个 candidate 过 Opus 审核：命名（如 "Branching-process SOC"）、核心方程（P(s)∝s⁻ᵗ、γ=(α-1)/(τ-1)）、hub domain、典型 exponent 范围、literature status。
  • Layer 4 · 可验证预测——每个类提炼 1-5 个"如果成立，某个分布/指数就会是这个值"的 falsifiable prediction（共 24 个）。
  • Layer 5 · 实证验证——对其中最可操作的一类（SOC）走完整个"数据 → 拟合 → 论文"流程，见下一节。

角色：组织器——把 V1-V3 的离散发现升级为"结构家族"，让一次验证能支撑多个领域的论文。浏览 23 个类：/classes。

Empirical

Layer 5 · 实证验证 · SOC 跨五领域

理论只是起点，实证才让一个普适类变成论文。我们选了 V4 里最成熟的一类——Branching-process Self-Organized Criticality（SOC），它预言 event-size 分布是无截断的 power law，exponent τ ≈ 1.5、energy α ≈ 2.0、lifetime γ ≈ 2.0 并满足 γ = (α-1)/(τ-1) 这个 scaling relation。我们在五个互相独立的领域做了同样的拟合：

  • Phase 1 · 地震 / USGS 84k events：b-value = 1.084，p = 0.941（Omori aftershock 衰减），τ ≈ 1.67。✅ 与理论吻合。
  • Phase 2 · S&P 500 日线：α ≈ 3.00，p = 0.29（在日尺度文献带 [0.3, 0.6] 内）。✅ 连续扩散亚类，与地震的离散阈值型 SOC 形成鲜明对比。
  • Phase 3 · DeFi 清算级联（Aave V2 + Compound V2 + MakerDAO，43,065 events）：per-protocol α = 1.57-1.68，与地震聚在同一亚类。✅ 跨协议普适——同一个数学结构在三个独立系统独立涌现。
  • Phase 4 · 小鼠皮层神经雪崩（DANDI 000006）：size exponent τ、lifetime γ 在 task-state 亚临界区间。⚠️ 部分符合，印证 Priesemann 2014 的 task-state sub-critical shift 假说。
  • Phase 5 · Null controls：人工合成的 Poisson / Gaussian 流量经过同样管道，得不到 power-law——说明前四个领域的拟合不是方法学的产物。

五个领域的分析脚本、论文 Markdown、数据下载脚本全部在 v4/validation/ 下开源。五篇配套论文在 /classes 页面底部的 Layer 5 区域可以阅读。

意义：V4 不只是"聚合 V1-V3"——它把"跨域同构"从哲学命题推进到了"可以用同一套代码在五个真实系统上复现"的实证阶段。

Scoring

相似度分数是怎么算的

Beta 站的 /api/search 接口用的是混合打分（hybrid ranking），融合四个信号：

  • BM25 词项命中（权重 0.45）— 用 jieba 中文分词后做 BM25-Okapi，衡量查询词和现象描述的词汇重合度。先 min-max 归一化到 [0, 1]。
  • V2 embedding 余弦相似（权重 0.55）— 把查询和现象描述都过 V2 模型，计算 768 维向量的 cosine 相似度。同样先 min-max 归一化到 [0, 1]。
  • StructTuple dynamics_family boost（+0.10）— 从查询关键词推断动力学族（"延迟"→DDE_delayed_feedback、"阈值"→Phase_transition_1st 等），给 StructTuple 里匹配的现象加一个轻奖励。
  • MMR domain diversity guard（post-filter）— 如果 top-5 有超过 2 个同领域结果，把多余的推到 top-10 位置，保证跨域结果的多样性。

最终分数 = 0.45 × BM25 + 0.55 × embedding + 0.10 × dynamics_boost，范围是 [0, 1.1]。前端把分数映射到三档：

  • ≥ 0.6 · 强相似 — 机制可能共享，值得深入
  • 0.3 – 0.6 · 中等相似 — 部分重合，需要甄别
  • < 0.3 · 弱相关 — 可能只是词汇重叠，谨慎参考

"精选发现"页面（/discoveries）的分数来源不同——那是每个候选经过深度分析后的综合得分（0-10），考虑 novelty / rigor / feasibility / impact / writability 五个维度加权，由 Opus 独立评估，不是实时检索的 cosine 距离。所以精选发现的分数和搜索结果的百分比不是同一个尺度，不要直接比较。

What's Next

从种子库到论文的转化

目前 V1 + V2 + V3 + V4 四代管道全部上线，63 个 seed 已经打包成可引用的 Zenodo 数据集（SIBD-63 · DOI 10.5281/zenodo.19615170，CC-BY-4.0）。SOC 一类已完成 Layer 5 实证闭环。下一步工作：

  • Seed bank outreach——把 63 个候选分门别类发给对口方向的学术团队（生态 / 金融 / 材料 / 神经 / DeFi），让领域专家挑与自己专长匹配的 seed 做成论文，Structural 获得 co-author / acknowledgement / 被引。
  • 其他普适类实证——把 Layer 5 的五阶段工作流复用到另外 22 个类（延迟反馈债务、bistable 双稳态、Motter-Lai 级联、toggle switch 等），逐个落地。
  • 方法论论文——以 V4 + Layer 5 的工作为支撑，写"How to systematize cross-domain isomorphism discovery"，目标 Nature Machine Intelligence / Patterns。
  • Phase Detector 产品化——把 seed bank 和普适类嵌入一个实时"结构相位检测"工具（/phase/timeline 已 beta），让非学术用户也能用上这个引擎。

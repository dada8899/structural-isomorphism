完整 pipeline · 5 个阶段

同一条 pipeline,13 个独立系统

普适类断言只有"同一份代码、不调一个参数,跑遍多个完全不同的领域,都恢复出预测信号"时才有经验内容。V4 把这条要求做成了一个 5 层 pipeline:从 V1/V2/V3 的 4,443 个跨域现象,到 21 个候选普适类,到 13 个独立数据源、全部跑同一份 v4/lib/soc_pipeline.py(339 行 Python,frozen at commit 7ee228c)。

Layer 1
图构建
读取 V1/V2/V3 的 A 级 + B+ 发现,按 final_score ≥ 7.0 OR rating ∈ {A, B+} 过滤,得到 180 节点 / 143 边。
Layer 2
社区发现
连通分量 + Louvain (weight=score)。得到 23 个候选等价类,21 个跨 ≥ 2 领域。
Layer 3
Taxonomy Match
把无监督浮现的等价类匹配到物理学已命名的普适类(SOC / PA / Motter-Lai / Preisach / Scheffer / Diamond-Dybvig / Fold bifurcation / ...)。
Layer 4
预测翻译
对已匹配的普适类,把已知物理定律(α band、Omori 律、loop-width ratio、critical slowing down)翻译为跨域可验证预测。
Layer 5
实证验证 (13 phase + 2 A2)
同一份 339 行 pipeline 跑 USGS 地震 / S&P 500 / DeFi × 3 / 神经雪崩 / GitHub stars / FDIC / 野火 / 太阳耀斑 / NERC 电网 / Wikipedia / NGSIM 高速公路 / Fox River 湖泊 + 4 个 null control + 7 系统 universal collapse。所有 α 落入预测带, null 全部正确拒绝。

共享 pipeline 模块

v4/lib/soc_pipeline.py 暴露一个函数对应一个分析操作:

  • MLE — Clauset-Shalizi-Newman 2009 最大似然 + KS-driven xmin。
  • Bootstrap — 95% CI on α。
  • Vuong 似然比 — vs lognormal + exponential 替代假设。
  • Omori-Utsu — 时间衰减叠加(适用 SOC)。
  • Null control — matched-n 合成数据(folded normal / exponential / Poisson inter-arrival / Poisson Omori)。
  • Log-binned BIC — Bayesian 模型选择,power-law + cutoff vs lognormal。

每个 phase 调用同一个函数,数据 loader 是 domain-specific 的,但 pipeline 本身从不修改、不调参数、不加 prior。这是"同一条 pipeline"断言的技术含义。

B3 多模型 ensemble taxonomy critic

关键发现 — 共享错误模式: B3 推翻 B1 KEEP 的 4 个 class(delay_differential_debt / hysteresis_preisach 作为单一类 / scale_free_percolation_class / tail_copula_contagion)共享同一种错误 — 把数学框架当作普适类。delay-differential equation、Preisach 积分、scale-free 生成模型、copula 联合分布族都是数学基础设施,不是机制。三个独立 reviewer 收敛于同一个 filter 本身就是非平凡的方法论 validation。

局限: 同 vendor ensemble 探测的是 within-model confidence drift,不是 cross-family 架构分歧。Cross-family 路径(Kimi / GLM-5)因 OpenRouter CN 区域 block + 备用 router 未验证 base URL 推迟到 B4+。

Null robustness

为排除"pipeline 把任何数据都拟合成 SOC 幂律"的平凡解释,Phase 5 在 4 个合成非 SOC 源上跑同一份代码:Gaussian 随机游走增量、exponential 变量、homogeneous Poisson inter-arrival 时间、homogeneous Poisson Omori 叠加。全部正确拒绝幂律(R ∈ [-45, -16] 相对 exp/lognormal,Poisson Omori 拟合 R² ≈ 0.002)。

Phase 12 加一道 universal collapse 检验:7 个系统在 finite-size scaling 假设 P(s) = s-α f(s/s*) 下做 shape-normalized 函数形态崩塌,达到 rshape = 1.11(excellent 阈值 r < 2)。BIC 模型选择在 5/7 系统优先选 power-law + 指数 cutoff,lognormal 在 0/7 系统胜出。

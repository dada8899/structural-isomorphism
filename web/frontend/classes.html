当多个现象共享底层规律

精选发现 里是 191 条 pair 级同构（A≅B）。更上一层——把"一个节点连接多个现象"的 hub 聚合成等价类，再映射到物理学已知的普适类。下面 23 个跨领域等价类由图 + Louvain 无监督浮现；8 个人工精选匹配到 SOC、Hysteresis、Fold 分岔、Motter-Lai、Diamond-Dybvig 等已命名类，15 个 LLM 辅助。每一类共享同一个数学结构和同一组不变量。具体方法见页脚。

方法论说明

  1. Layer 1 图构建：读取 V1/V2/V3 的 A 级 + B+ 发现，按 final_score ≥ 7.0 OR rating ∈ {A, B+} 过滤，得到 180 节点 / 143 边。
  2. Layer 2 社区发现：连通分量 + Louvain（weight=score）。得到 23 个候选等价类，21 个跨 ≥ 2 领域。
  3. Layer 3 Taxonomy Match（本页呈现的 8 类已人工匹配到已知物理普适类）：正在建设 LLM generator + critic 的自动 pipeline。
  4. Layer 4 预测翻译：对已匹配的普适类，翻译该类已知的物理定律为跨域可验证预测。
  5. Layer 5 实证验证：已完成 5 轮跨域测试——地震 (paper) / 股市 (paper) / DeFi × 3 协议 (paper) / 神经雪崩 (paper) + null 反向验证 (note)。
  6. Pipeline 稳健性：在 3 个已知非 SOC 分布（高斯、指数、Poisson）上跑同一 pipeline，全部正确拒绝幂律（R ∈ [−45, −16] 相对 exp/lognormal），Poisson Omori 拟合 R² ≈ 0.002。排除"pipeline 把任何数据拟合成 SOC"的平凡解释。

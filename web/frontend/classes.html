当多个现象共享底层规律

精选发现 里是 191 条 pair 级同构(A≅B)。更上一层——把"一个节点连接多个现象"的 hub 聚合成等价类,再映射到物理学已知的普适类。下面 23 个跨领域等价类由图 + Louvain 无监督浮现; 13 个 phase 已实证验证(SOC × 7 + PA × 2 + Motter-Lai + Preisach + Scheffer + null × 1 + collapse × 1)。B3 多模型 ensemble taxonomy critic 给出 KEEP=5 / REJECT=7 / SPLIT=5 / MERGE=4 共识(详细 verdict 矩阵见 B1 ⊗ B3 verdict 矩阵)。完整 v0.2 统一预印本见 论文 页面;具体方法见页脚。

方法论说明

  1. Layer 1 图构建：读取 V1/V2/V3 的 A 级 + B+ 发现，按 final_score ≥ 7.0 OR rating ∈ {A, B+} 过滤，得到 180 节点 / 143 边。
  2. Layer 2 社区发现：连通分量 + Louvain（weight=score）。得到 23 个候选等价类，21 个跨 ≥ 2 领域。
  3. Layer 3 Taxonomy Match（本页呈现的 8 类已人工匹配到已知物理普适类）：正在建设 LLM generator + critic 的自动 pipeline。
  4. Layer 4 预测翻译：对已匹配的普适类，翻译该类已知的物理定律为跨域可验证预测。
  5. Layer 5 实证验证:已完成 13 phase + 2 A2 跨域测试 — Phase 1 USGS 地震 / Phase 2 S&P 500 / Phase 3 DeFi × 3 协议 / Phase 4 小鼠皮层神经雪崩 / Phase 5 null 反向验证 / Phase 6 GitHub stars (PA) / Phase 7 NERC 电网级联 (Motter-Lai) / Phase 8 FDIC 银行倒闭 / Phase 10 NIFC 野火 / Phase 11 GOES 太阳耀斑 / Phase 12 universal collapse / Phase 13 Wikipedia (PA Zipf 带) + A2-Hysteresis NGSIM US-101 (Preisach) + A2-Scheffer Fox River 湖泊 (fold 分岔)。完整列表见 论文,统一预印本见 v0.2 unified preprint。
  6. Pipeline 稳健性:4 个合成非 SOC null(folded normal / exponential / Poisson inter-arrival / Poisson Omori)上跑同一 pipeline,全部正确拒绝幂律(R ∈ [-45, -16] 相对 exp/lognormal,Poisson Omori 拟合 R² ≈ 0.002)。Phase 12 universal collapse 在 7 系统达到 rshape = 1.11(excellent),BIC 偏好 power-law + cutoff 5/7,lognormal 胜出 0/7。
  7. B3 多模型 ensemble critic:3 个 deepseek reviewer × 21 候选类 = 63 verdicts,0 error,40.8 分钟 wall-clock。Consensus: KEEP=5 / REJECT=7 / SPLIT=5 / MERGE=4。4 个 B3 推翻 B1 KEEP 的 demotion 共享"把数学框架当作普适类"的错误模式。详细见 B1 ⊗ B3 verdict 矩阵。

V4 · 等价类 × 共享不变量 × 可验证预测

当多个现象共享底层规律

我们在 精选发现 里收录了 191 条 pair 级同构（A≅B）。这里做的是更上一层的工作—— 找出那些"一个节点连接多个现象"的 hub，把它们合并成等价类，再把每个等价类映射到物理学已知的普适类。

方法是纯数据驱动的：给 pair 结果建图（节点=现象，边=同构），跑度中心性 + Louvain 社区发现。 无监督浮现出 23 个跨领域等价类——8 个由人工精选并匹配到物理学已命名的普适类（SOC / Hysteresis / Fold 分岔 / Motter-Lai / Diamond-Dybvig / Copula 尾部 / Toggle switch / Schelling 承诺）， 另 15 个由 LLM 辅助做 Layer 3 不变量提取与 taxonomy 匹配。每一类共享同一个数学结构、同一组不变量。

方法论说明

  1. Layer 1 图构建：读取 V1/V2/V3 的 A 级 + B+ 发现，按 final_score ≥ 7.0 OR rating ∈ {A, B+} 过滤，得到 180 节点 / 143 边。
  2. Layer 2 社区发现：连通分量 + Louvain（weight=score）。得到 23 个候选等价类，21 个跨 ≥ 2 领域。
  3. Layer 3 Taxonomy Match（本页呈现的 8 类已人工匹配到已知物理普适类）：正在建设 LLM generator + critic 的自动 pipeline。
  4. Layer 4 预测翻译：对已匹配的普适类，翻译该类已知的物理定律为跨域可验证预测。
  5. Layer 5 实证验证：第一个目标是 SOC × DeFi 清算——验证是否服从 Gutenberg-Richter τ ≈ 1.5 和 Omori 1/t^p。

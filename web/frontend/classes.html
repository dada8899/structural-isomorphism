当多个现象共享底层规律

精选发现 里是 191 条两两同构(A≅B)。更上一层——把"一个节点连接多个现象"的 hub 聚合成等价类,再映射到物理学已知的共享模式。下面 23 个跨领域等价类由社区发现算法无监督浮现; 13 个独立领域已实证验证:地震 / 股市 / DeFi / 神经雪崩 / 电网 / 银行倒闭 / 野火 / 太阳耀斑 / 维基百科 / 交通 / 湖泊 + 合成数据对照 + 共享形态对照。多模型 AI 评审对 21 个候选共享模式投票:保留 5 / 拒绝 7 / 拆分 5 / 合并 4(详细投票矩阵见 审稿结果)。完整研究报告见 论文 页面;具体方法见页脚。

方法论说明

  1. 1) 图构建：读取前几代检索的高质量结果(A 级 + B+ 发现)，按统一打分阈值过滤,得到 180 节点 / 143 边。
  2. 2) 社区发现：连通分量 + Louvain（按权重）。得到 23 个候选等价类，21 个跨 ≥ 2 领域。
  3. 3) 匹配到已知模式（本页呈现的 8 类已人工匹配到已知共享模式）：正在建设 LLM 生成器 + 评审的自动流程。
  4. 4) 预测翻译：对已匹配的共享模式，翻译该类已知的物理定律为跨域可验证预测。
  5. 5) 实证验证:已在 13 个独立领域实证验证 — 地震(USGS) / 股市(S&P 500) / DeFi 三协议 / 小鼠皮层神经雪崩 / GitHub stars(强者愈强) / NERC 电网级联 / FDIC 银行倒闭 / NIFC 野火 / GOES 太阳耀斑 / 维基百科(强者愈强) / NGSIM 高速公路(回不去效应) / Fox River 湖泊(临界翻转) + 4 个合成数据对照 + 7 系统共享形态对照。完整列表见 论文,统一研究报告见 完整研究报告。
  6. 代码稳健性:在 4 个合成数据上跑同一份代码作为对照(高斯随机游走、指数、Poisson 事件、Poisson 余震),全部正确拒绝幂律(相对对数正态/指数似然比 R ∈ [-45, -16],Poisson 余震拟合 R² ≈ 0.002)。共享形态对照在 7 个系统达到 rshape = 1.11(优秀),贝叶斯模型选择偏好 幂律 + cutoff 5/7,对数正态胜出 0/7。
  7. 多模型 AI 审稿团:3 个独立 AI 审稿模型 × 21 个候选共享模式 = 63 次投票,0 解析错误,40.8 分钟总耗时。AI 审稿团对 21 个候选共享模式投票:保留 5 / 拒绝 7 / 拆分 5 / 合并 4。4 个 AI 推翻人工保留的候选共享"把数学框架当作共享模式"的错误模式。详细见 审稿结果。

# A2-Hysteresis — Traffic Flow Verdict (2026-05-13)

**Phase.** Layer 5 A2 (first attempt at a non-SOC class).
**Class under test.** `hysteresis_preisach` — V4 taxonomy class #2.
**Domain.** Highway traffic flow, NGSIM US-101 South (2005-06-15, 7:50-8:35am) + literature anchors.

---

## Predicted invariants (Layer 4)

| signature | quantity | predicted band |
|---|---|---|
| (i) loop width | q_c1 / q_c2 ratio between upper free-flow capacity and lower congested-branch capacity | [1.25, 1.55] |
| (ii) first-order character | sharp speed transition v ≥ 60 km/h → v ≤ 40 km/h at fixed location; meta-stable corridor traversed briefly | qualitative |

## Measurement

| signature | source | result |
|---|---|---|
| (i) loop width — empirical | NGSIM US-101, n = 4,538 cells (lane × 30 s × 200 ft, Edie aggregation, q ≤ 2,800 cap) | q_c1 = 2,105 / q_c2 = 2,273 → ratio **0.926**, 95 % CI [0.897, 0.975]. **Outside band** — dataset structurally captures only the loading half of the cycle. |
| (i) loop width — Treiber 2013 chap. 8 A5 | textbook calibration | 2,200 / 1,600 = **1.375** — inside [1.25, 1.55] |
| (i) loop width — Geroliminis 2008 Yokohama | published MFD | 18 / 13 = **1.385** — inside [1.25, 1.55] |
| (ii) first-order signature — empirical | NGSIM, 55 monitored locations (lane × space-bin, ≥ 30 time bins each) | 25/55 locations (**45 %**) exhibit free → jam transition; median crossing duration **630 s** (10.5 min of a 45-min window) |

## Composite verdict

**CONFIRMED_COMPOSITE.**

The first-order discontinuity signature is empirically confirmed on NGSIM US-101: 45 % of monitored locations show the sharp speed transition consistent with first-order behaviour, well above the 30 % threshold. The loop-width ratio q_c1 / q_c2 cannot be measured end-to-end from a 45-minute peak-hour-loading-only dataset — the recovery half of the hysteresis cycle is absent. Both independent literature anchors (Treiber & Kesting 2013, Geroliminis & Daganzo 2008) give ratios ≈ 1.38, squarely inside the predicted [1.25, 1.55] band and in 1 % agreement with each other.

Verdict is qualified by being a **hybrid (empirical first-order + literature loop-width)** rather than a single self-contained empirical loop measurement.

## Honest caveats

1. The NGSIM-only loop-width ratio (0.926) is *outside* the predicted band. We do not treat this as class rejection because the dataset structurally lacks the congestion-recovery half-cycle needed to estimate the lower critical flow q_c2 in the canonical hysteresis sense. The "congested branch" cells in NGSIM are bottleneck-stable-discharge cells, whose q approaches free-flow capacity by a well-known stylised fact, hence the ratio ≈ 1.
2. This is V4 taxonomy class #2 (the *second* class to be empirically tested) and the *first* non-SOC class. The pipeline used here (branch-percentile-ratio + temporal-discontinuity-scan) is structurally different from the A1 SOC stack (Clauset MLE + Omori-Utsu) and is calibrated to a different invariant family.
3. A future phase using Caltrans PeMS archival multi-day data (after registering for account access) would close the empirical loop-width gap and upgrade the verdict from CONFIRMED_COMPOSITE to CONFIRMED.
4. The first-order signature scan threshold (30 % of locations with detected transitions) is a design choice, not a hard physical bound. A stricter threshold (e.g. 60 %) would not have been met by NGSIM. The result is moderately sensitive to this choice.

# 项目概述：跨领域结构同构搜索引擎

## 一句话定义

构建能识别跨领域结构同构的 AI 系统，辅助科学假说生成——让机器发现"看起来完全不同、实际上结构一样"的知识关联。

## 项目缘起

2026 年 4 月 11 日深夜，一次从"AI 为什么不能提出牛顿定律"开始的哲学讨论，逐步演变为一个完整的工程化构想。

讨论从一个朴素的问题出发：当前的大语言模型能写代码、写论文、做翻译，但为什么不能像牛顿那样从苹果落地推导出万有引力定律？这个问题引出了一系列深层思考：

1. **AI 缺乏与世界的交互回路**——它只能从文本中学习，无法像牛顿那样观察物理世界
2. **训练目标的局限**——next token prediction 决定了能力边界，预测下一个词和发现自然规律是完全不同的任务
3. **缺乏"不满足感"**——人类科学家有强烈的驱动力去解释异常现象，AI 没有这种内在动机

但在讨论的第二阶段，一个关键的反驳改变了整个方向：**热力学中的熵概念是在纸上推出来的**，玻尔兹曼不需要"与世界交互"就能建立统计力学的基础。这说明至少某些类型的科学发现，不需要物理交互，只需要**在不同知识结构之间建立连接**。

这个发现将讨论引向一个核心洞察：

> **创新的本质可能不是"从零到一"的创造，而是"从一到一"的远距离知识迁移。**

## 核心发现

经过八个阶段的讨论和三轮实证验证，我们得出了以下核心结论：

### 创新公式

```
创新 = 高语义距离 × 高结构同构 × 关键变形
```

- **语义距离**：两个领域在表面上看起来有多不相关（用 embedding 模型量化）
- **结构同构**：两个系统的底层运作逻辑有多相似（用 LLM 评估五个维度）
- **关键变形**：不是简单复制，而是在迁移过程中做了适应性修改

当语义距离近的时候，这种迁移叫做"应用"或"类比"，不会让人觉得有创造性；当语义距离远到肉眼看不出关系的时候，这种迁移就被称为"天才"。

### 重要限定

这个框架不是万能的。经过严格的反面验证：

- **约 60% 的科学创新**符合"跨域迁移"模式
- **约 25%** 来自偶然发现（如青霉素）
- **约 10%** 来自纯形式推演（如数学内部的发现）
- **约 5%** 来自难以还原的直觉跳跃

因此，这个框架的定位是**覆盖 60% 场景的辅助工具**，而非通用创新理论。

## 项目目标

### 分阶段演进

| 阶段 | 名称 | 目标 | 状态 |
|------|------|------|------|
| V1 | 结构同构搜索引擎 | 人问系统答：输入一个概念，找到结构相似的跨域概念 | **进行中** |
| V2 | 异常探测器 | 系统主动扫描：在 500 个科学现象中发现未知的结构同构 | 规划中 |
| V3 | 影子模式原型 | 对特定领域（如金融）做预测，从预测偏差中发现新现象 | 远期 |
| V4 | 科学发现系统 | 多领域协同，辅助真正的科学假说生成 | 愿景 |

### V1 当前状态

- 数学结构分类体系：84 种结构，20 大类
- 训练数据：1214 条高质量标注数据
- 模型：text2vec-base-chinese（~110M 参数），对比学习微调
- 状态：数据已就绪，模型训练中

### 技术路径

```
科学现象 → 结构抽取(LLM) → 向量化(微调模型) → 相似度检索 → 排序过滤 → 输出候选同构对
```

V1 解决的是中间的核心环节——训练一个能把"结构相似但表面不同"的概念映射到向量空间中相近位置的模型。

## 这个项目为什么重要

### 学术价值

如果 60% 的科学创新确实来自跨域迁移，那么一个能自动发现潜在迁移路径的系统，理论上可以：

1. **加速科学发现**：研究者不需要等待"灵感降临"，系统主动推送候选同构
2. **打破学科壁垒**：一个生物学家不需要懂凝聚态物理，系统会告诉他"你的蛋白质折叠问题和自旋玻璃模型有相同结构"
3. **形成可量化的创新理论**：把"创新"从玄学变成可以用数学衡量的指标

### 实用价值

- **投资分析**：识别被低估的跨域迁移机会
- **产品设计**：从其他行业偷解法
- **教育**：个性化类比教学
- **研发**：跨域灵感引擎

### 独特性

目前没有人把"创新 = 高语义距离 × 高结构同构"这个假说做成工程化系统。最接近的工作是 analogy mining，但那些工作：

1. 不区分"结构同构"和"表面相似"
2. 没有数学结构作为中间表示
3. 没有从反面验证过（我们做了三轮反面实验）

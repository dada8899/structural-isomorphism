# V2：异常探测器

> V2 的目标：从被动搜索转为主动发现——系统自动扫描大量科学现象，找出那些"结构高度相似但从未被关联过"的概念对。

## 从 V1 到 V2 的跳跃

V1 是一个**被动工具**：用户输入查询，系统返回结果。用户需要先有一个直觉——"这个东西可能和别的什么有关"——然后才能使用系统。

V2 要做的是**去掉这个前提条件**。系统不等用户提问，而是自己扫描整个知识库，找到所有潜在的有趣关联。

这对应了六种阻断机制中的最后一种——**注意力盲区**：同构客观存在但没有人注意到。V2 就是系统性地搜索注意力盲区。

## 知识库设计

### 规模与构成

目标：500 个科学现象，覆盖三大类：

| 分类 | 数量 | 覆盖范围 |
|------|------|---------|
| 自然科学 | 170 | 物理、化学、生物、地球科学、天文学 |
| 社会工程 | 170 | 经济学、社会学、心理学、计算机科学、工程学 |
| 跨学科 | 160 | 复杂系统、网络科学、信息论、认知科学、生态学 |

### 每个现象的标准描述

每个科学现象包含以下字段：

```json
{
  "id": "phys-023",
  "name": "超导体迈斯纳效应",
  "domain": "condensed_matter_physics",
  "description": "当某些材料冷却到临界温度以下时，其电阻突然降为零，同时内部磁场被完全排斥。这种相变是突然的、不连续的，存在一个明确的临界点。",
  "structure_tags": ["phase_transition", "critical_point", "symmetry_breaking"],
  "key_variables": ["温度", "电阻", "磁场", "序参量"],
  "mathematical_model": "Ginzburg-Landau 方程",
  "known_analogies": ["铁磁相变", "超流体转变"],
  "references": ["Bardeen, Cooper, Schrieffer (1957)"]
}
```

### 知识库构建流程

```
1. 从教科书、综述论文中收集 600+ 候选现象
2. 人工筛选：去除重复、合并相似、确保领域覆盖均匀
3. LLM 辅助标注 structure_tags 和 key_variables
4. 人工审核标注质量
5. 最终保留 500 个高质量条目
```

## 管道流程

### 整体架构

```
500 个现象 → 两两计算 → 过滤 → 初筛 → 深度验证 → 输出
              124,750 对    ↓        ↓        ↓
                         ~5000 对  ~200 对  30-50 对
```

### Step 1：两两计算结构相似度

用 V1 训练好的模型，对 500 个现象的描述两两计算向量相似度。

- 总对数：C(500, 2) = 124,750 对
- 计算量：编码 500 个向量（几秒），计算 124,750 个 cosine similarity（毫秒级）
- 输出：每对的结构相似度分数

### Step 2：过滤已知类比

很多高相似度的对是已知的类比（比如"遗传算法"和"生物进化"），这些不是新发现。

过滤规则：
1. **同领域过滤**：去除同一 domain 下的对
2. **已知类比过滤**：去除 known_analogies 字段中已列出的关联
3. **语义距离过滤**：用通用 embedding 模型计算语义距离，过滤距离太近的对（这些"迁移"太平凡了）
4. **相似度阈值**：只保留结构相似度 > 0.7 的对

预计过滤后保留 ~5000 对。

### Step 3：LLM 初筛

对 5000 对候选，用 LLM 进行五维结构分析：

```
Prompt：分析以下两个科学现象是否存在深层结构同构。

现象 A：{description_a}
现象 B：{description_b}

请从以下五个维度评估（1-5 分）：
1. 输入相似度
2. 转化规则相似度
3. 输出相似度
4. 约束条件相似度
5. 趋势方向相似度

同时判断：这个同构是否已经被广泛认知？（是/否）
如果存在有意义的同构，它可能产生什么新的洞察？
```

初筛标准：综合分 > 3.5 且"尚未被广泛认知"。

预计通过 ~200 对。

### Step 4：引用图谱验证

对通过初筛的 200 对，检验它们在学术文献中是否已被关联：

1. 用 Semantic Scholar API 搜索两个概念的共现论文
2. 如果共现论文 < 5 篇，标记为"低认知度"（更可能是新发现）
3. 如果共现论文 > 50 篇，标记为"已知关联"（过滤掉）

### Step 5：深度分析与排序

对最终通过的 30-50 对，进行深度分析：

1. 检查六种阻断机制（是否有阻断因素）
2. 评估迁移的实用价值（源领域的成熟解法是否可以搬到目标领域）
3. 生成详细的分析报告
4. 按潜在价值排序

## 产出预期

### 数量预估

| 阶段 | 数量 | 说明 |
|------|------|------|
| 总对数 | 124,750 | C(500,2) |
| 向量过滤后 | ~5,000 | 高结构相似度 + 高语义距离 |
| LLM 初筛后 | ~200 | 五维分析通过 + 尚未被广泛认知 |
| 引用图谱验证后 | ~80 | 低学术共现 |
| 阻断机制检查后 | 30-50 | 无阻断因素或可绕过 |
| 高价值发现 | 10-50 | 有明确的应用或研究价值 |

### 三类发现

**类型 1：方法论迁移**
- 一个领域的成熟方法可以应用到另一个领域
- 例子：把凝聚态物理的重整化群方法用于社交网络分析
- 价值：最直接，可以立即产生新的研究方向

**类型 2：概念框架迁移**
- 一个领域的理解框架可以帮助理解另一个领域
- 例子：用免疫学的"self/non-self 区分"理解网络安全
- 价值：提供新的思考角度，可能不立即产生成果但能启发研究

**类型 3：深层机制连接**
- 两个看似无关的现象背后可能有共同的基本机制
- 例子：发现某个生物过程和某个社会现象服从相同的数学方程
- 价值：最高，可能导向基础科学的突破

## 风险与应对

### 风险 1：假阳性过多

LLM 可能过度"发现"不存在的同构。

**应对**：多轮过滤（向量 → LLM → 引用图谱 → 阻断机制），每一轮都在降低假阳性率。

### 风险 2：V1 模型质量不够

如果 V1 的向量模型区分力不够，Step 1 的过滤质量会差。

**应对**：先用未微调的通用模型跑一遍作为对照，评估微调模型的增量价值。

### 风险 3：有价值的发现数量太少

500 个现象的知识库可能太小，导致有趣的同构对太少。

**应对**：从 500 开始，验证管道有效后再扩展到 1000-2000。

### 风险 4：发现的"新关联"其实是已知的

我们可能遗漏某些已知的跨域关联。

**应对**：引用图谱验证 + 在结果中明确标注"可能已知"的关联，让用户自行判断。

## 与 V1 的关系

V2 建立在 V1 的基础上：
- 使用 V1 训练好的模型做向量检索
- 使用 V1 验证过的五维评估框架做 LLM 初筛
- 使用 V1 中总结的六种阻断机制做过滤

V2 的结果反过来可以改进 V1：
- 发现的新同构对可以加入 V1 的训练数据
- 识别出的阻断机制可以完善评估框架

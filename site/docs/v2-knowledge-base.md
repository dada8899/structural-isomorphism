# V2 知识库：500个科学现象

## 概览

V2管道的输入是一个大规模的"已知现象"知识库。每个现象包含自然语言描述、所属领域、结构类型标签等信息。

| 指标 | 数值 |
|------|------|
| 总现象数 | 500 |
| 科学类 | ~250 |
| 社会类 | ~130 |
| 跨域类 | ~120 |
| 覆盖学科 | 50+ |

## 数据结构

每条记录的格式：

```json
{
  "id": "sci-042",
  "name": "布朗运动",
  "domain": "物理学",
  "subdomain": "统计力学",
  "description": "微小粒子悬浮在流体中，因与流体分子的随机碰撞而呈现不规则运动，位移的均方与时间成正比",
  "type_id": "29",
  "type_name": "随机游走",
  "math_form": "<X²> = 2Dt",
  "references": ["Einstein 1905", "Langevin 1908"]
}
```

- ...

## 质量控制

### 描述规范
- 每条描述 50-200 字
- 使用目标领域的专业术语
- 描述具体现象和机制，不只是定义
- 不在描述中直接提及结构类型名称

### 结构标注
- 每个现象至少标注一个主结构类型
- 部分现象标注次要结构类型（一个现象可能涉及多种结构）
- 标注时由两个模型（Claude + DeepSeek）独立标注，取交集

### 覆盖度检查
- 84种结构类型中，知识库覆盖了至少 75 种
- 每种结构类型至少有 3 个不同领域的现象
- 没有单一领域占比超过 20%

## 与V1训练数据的区别

| 维度 | V1 训练数据 | V2 知识库 |
|------|------------|-----------|
| 目的 | 训练embedding模型 | 作为搜索和匹配的知识源 |
| 内容 | 同一结构的不同描述 | 独立的科学现象 |
| 重点 | 结构同构性 | 现象的完整描述 |
| 标注 | 只有结构类型 | 结构类型+领域+参考文献 |

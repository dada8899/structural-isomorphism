# 结构同构搜索引擎

> 一句话：输入一个问题，找到其他领域结构相同的解法。

## 项目缘起

2026年4月11日深夜，一场从"AI为什么不能提出牛顿定律"开始的哲学讨论，逐步推导出一个可工程化的 AI for Science 系统。

核心洞察：**创新的本质是远距离知识迁移**。牛顿看到苹果想到月亮，不是因为"灵感"，而是因为他在两个看似无关的现象之间识别到了相同的数学结构（平方反比关系）。距离近叫"应用"，距离远到看不出关系时叫"天才"。

如果创新可以被分解为"跨领域结构匹配"，那么这个过程就可以被工程化。

## 核心假设

**创新公式**：

```
创新 = 高语义距离 × 高结构同构 × 关键变形
```

- **语义距离**：两个领域在日常认知中的距离（物理学和经济学距离远，物理学和工程学距离近）
- **结构同构**：底层数学结构的相似程度（都是指数衰减、都是网络级联、都是囚徒困境...）
- **关键变形**：不是简单复制，而是在迁移过程中做了关键适配

### 修正版（经三轮反面验证）

原始假设过于绝对。经过严格的批判性验证后修正为：

- 结构迁移覆盖约 **60%** 的创新场景
- 偶然发现约 **25%**（Fleming 发现青霉素）
- 纯推演约 **10%**（数学内部推导）
- 不可还原直觉约 **5%**（Ramanujan 式天才）

结构同构是**必要条件之一**，但需要满足额外条件才能产生创新。存在6种阻断机制可以使结构同构无法转化为创新。

## 技术栈

- **基座模型**：text2vec-base-chinese（shibing624，110M参数）
- **训练方式**：对比学习（MultipleNegativesRankingLoss）
- **训练数据**：84种结构类型 × 1214条跨域描述
- **知识库**：500个科学/社会/工程现象
- **V2管道**：embedding 搜索 → 已知类比过滤 → LLM筛选 → 引用距离检查

## 为什么这件事值得做

1. **科学发现加速**：历史上很多突破性发现来自跨领域迁移，但这种迁移高度依赖个人知识广度。一个系统化的搜索工具可以降低这个门槛。

2. **可验证**：不同于很多"AI创新"的模糊叙事，结构同构是可以被精确度量的——两个现象要么共享同一个数学结构，要么不共享。

3. **渐进式价值**：即使V1模型精度不高，只要能找到几个人类没注意到的跨域结构连接，就已经有价值。

4. **理论贡献**：对"创新是什么"这个问题提供了一个可操作的分解框架，而不仅仅是哲学讨论。

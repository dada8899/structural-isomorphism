## 详细数据

### Test 1：轮廓系数（Silhouette Score）

衡量同类描述是否聚在一起、不同类是否分开。范围 -1 到 1。

- 基线模型：**-0.01**（完全没有结构聚类能力）
- 微调后：**0.85**（非常清晰的结构聚类）
- 提升：**+0.86**

> 0.85 的轮廓系数意味着：同一种数学结构类型的描述几乎总是彼此最近邻，不同类型之间有明确的分界。这在 NLP 任务中是非常高的分数。

### Test 2：类内 vs 类间相似度

| | 基线模型 | 微调后 |
|---|---------|--------|
| 类内均值（同类型对）| 0.643 ± 0.075 | **0.933 ± 0.033** |
| 类间均值（不同类型对）| 0.569 ± 0.064 | **0.174 ± 0.153** |
| 差距 | 0.074 | **0.758** |

关键发现：
- 微调后，同类型描述的相似度从 0.64 提升到 **0.93**（几乎完全匹配）

- 差距从 0.07 扩大到 **0.76**——区分力提升了 **10 倍**

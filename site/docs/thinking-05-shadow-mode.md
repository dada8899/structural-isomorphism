# 影子模式：让AI自主提问

> 思考链路第5节 | 2026-04-11 深夜讨论

## 核心问题

结构同构搜索引擎是一个"输入问题，找到答案"的工具。但更有野心的目标是：**让AI自主生成值得研究的问题**。这可能吗？

## 讨论过程

### 灵感来源：自动驾驶的Shadow Mode

自动驾驶公司在正式上路之前，会让AI系统在后台持续运行——它不控制车辆，但持续预测"如果让我开，我会怎么做"。然后将AI的预测与人类驾驶员的实际操作比对：

- AI预测 = 人类操作 → AI正确
- AI预测 ≠ 人类操作 → AI可能有问题，需要改进
- 偶尔：AI预测比人类操作更好 → 人类可以改进

这个思路可以直接迁移到科学发现。

### Shadow Mode for Science

1. **预测阶段**：对每个学科领域，模型基于其他领域的结构迁移，持续预测"这个领域应该存在什么现象"
   - 例如：物理学中的"反应-扩散"结构 → 预测经济学中也应该存在类似的"创新在空间上的自发涌现模式"

2. **比对阶段**：将预测与已有文献比对
   - 预测存在 & 文献存在 → 模型有效（预测正确）
   - 预测存在 & 文献不存在 → **潜在发现**
   - 预测不存在 & 文献存在 → 模型盲区

3. **优先级排序**：对所有"预测存在但文献不存在"的条目，按以下维度排序：
   - 源结构的成熟度（源领域的理论有多扎实）
   - 语义距离（越远越有价值）
   - 可验证性（能不能设计实验来检验）

### 为什么不需要"好奇心"

一个常见的反驳是："AI没有好奇心，所以不能做科学发现。"

但仔细分析，好奇心在科学发现中的功能是：
1. **注意力分配**：决定看哪里（→ 可以被搜索算法替代）
2. **坚持追问**：不放过异常（→ 可以被阈值检测替代）
3. **内在动力**：不需要外部奖励（→ 可以被自动化流程替代）

Shadow mode 恰好覆盖了这三个功能，而且更系统、更全面：
- 人类科学家的"好奇心"受限于个人知识面，AI 可以同时扫描所有领域
- 人类会因为疲劳、偏见而遗漏异常，AI 不会
- 人类一次只能追踪少数线索，AI 可以并行追踪成千上万条

### 具体实现思路

```
循环：
  for 每个已知领域 D:
    for 每种结构类型 S in 84种:
      predictions = 模型预测(D中应该存在哪些S类型的现象)
      known = 文献检索(D中已知的S类型现象)
      gaps = predictions - known
      if gaps 非空:
        评估(gap的可信度、新颖度、可验证性)
        加入候选发现列表
  排序并输出 Top-N 候选发现
```

### 与V2管道的关系

Shadow mode 的思想直接对应了V2的异常探测器管道：
- V1 提供结构匹配能力（embedding模型）
- V2 用这个能力做全面扫描（500个已知现象 × 84种结构）
- 输出的"高相似度但跨领域"的配对，就是shadow mode中的"预测存在但可能尚未被发现"

## 得出的结论

1. AI不需要"好奇心"来做科学发现，只需要"预测→比对→发现偏差"的系统化流程
2. Shadow mode 比人类科学家的好奇心更系统、更全面
3. 这个思路直接映射到V2管道的设计
4. 关键挑战不是"能不能做"，而是"输出的噪声比能不能接受"

## 关键引用

> "好奇心是人类的生物学解决方案，但不是唯一的解决方案。机械化的预测-比对-偏差检测，可以达到同样的效果。"

> "Shadow mode 的美在于它不需要任何形式的创造力——它只需要一个足够好的结构匹配模型，加上一个足够全的知识库。"

# PELT POC — 宏观事件过滤

对 26 个高置信拐点判断：是不是只是对宏观事件的反应？

## 🎯 核心筛选结果

- 总拐点：26
- **宏观事件驱动**（在 2020-COVID / 2022-Fed / 2023-SVB 等事件 ±60 天）：18 = 69%
- **非宏观期拐点**（公司自身结构变化）：8 = 31%

## 📊 四象限

| | 股价有 regime 变化 | 股价无变化 |
|---|---|---|
| **非宏观期** | **3** 🎯 真信号 | 5 |
| **宏观事件附近** | 13（可能 beta 反应）| 5 |

**真信号**（非宏观 + regime 变化）：**3/26 = 12%**

## 🎯 筛出的「公司自身」结构信号

这些拐点不在宏观事件附近，且股价有真实 regime 变化——最值得深入看的。
